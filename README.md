# О программе
Утилита для копирования перемещения файлов на ANSI-C. 

# Использование

Копирование файла
```powershell
./fmv C:\file.txet + C:\new_catalog\copy.text
```
Перемещение файла
```powershell
./fmv C:\file.text - C:\new_catalog\file.text
```
Изменение имени и расширения
```powershell
./fmv C:\file.text - C:\new_name.text
```
